
Segarin - Full Static Website (Demo)
-----------------------------------

Files:
- index.html            -> Main page (open in browser)
- assets/logo.png       -> Provided Segarin logo
- assets/prod1.png ...  -> Generated placeholder product images

How to use:
- To preview locally: open 'index.html' in your browser.
- To publish: upload the contents of this folder to GitHub Pages, Netlify, or Vercel.
  * For GitHub Pages: push files to a repo and set GitHub Pages to use the main branch (root).
  * For Netlify: drag & drop the ZIP or the folder into Netlify drag & drop deploy.
